# Spot Mate

A Next.js project to discover, share, and chat about social hangout spots.

## Features
- User authentication with BetterAuth
- Spot creation and viewing
- Geolocation-based nearest spots
- Comments and chat with per-day limits using Redis
- Premium features with Razorpay integration
- Cursor-based pagination for scalable queries

## Technologies
- Next.js
- MongoDB with Mongoose
- Redis
- JWT authentication
- Razorpay
- TypeScript
